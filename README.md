# Fokir Portfolio Website

A responsive personal portfolio website built using pure HTML and CSS.

## 🚀 Features
- Responsive design using Media Queries
- Hero section with typing animation
- Portfolio gallery with overlay effects
- Blog section
- Client feedback card
- Contact form UI

## 🛠 Technologies Used
- HTML5
- CSS3
- Media Queries
- Font Awesome
- Google Fonts
- Typed.js
